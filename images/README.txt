📁 IMAGES KANSIO - Lisää tänne golf-meemikuvat

🎯 OHJEET:

1. Tallenna golf-meemit koneellesi (Pinterest, Reddit, Google, ym.)

2. Nimeä ne:
   - meme1.jpg
   - meme2.jpg
   - meme3.jpg
   - meme4.jpg
   jne...

3. Kopioi kuvat tähän kansioon

4. Päivitä sivu selaimessa (F5)

5. VALMIS! Kuvat näkyvät automaattisesti sivulla

💡 VINKKEJÄ:

- Tuetut formaatit: .jpg, .png, .gif, .webp
- Suositeltava koko: 800x600 tai suurempi
- Jos kuva ei lataudu, näkyy placeholder-kuva
- Voit muokata tekstit script.js tiedostossa

🚀 LISÄÄ MEEMEJÄ:

Jos haluat lisätä enemmän kuin 9 meemiä:
1. Avaa script.js
2. Lisää uusi meemi-objekti:

{
    image: "images/meme10.jpg",
    title: "Uusi meemi",
    caption: "Hauska teksti tähän"
}

Nauti! ⛳
